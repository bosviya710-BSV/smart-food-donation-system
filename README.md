# Smart Food Donation System

## Project Description
Smart Food Donation System is a web application that connects hotels, restaurants, NGOs, volunteers, and administrators to reduce food waste by donating surplus food to people in need.

## Technologies Used
- PHP
- MySQL
- HTML
- CSS
- JavaScript

## Features
- User Registration & Login
- Hotel Dashboard
- NGO Dashboard
- Volunteer Dashboard
- Admin Dashboard
- Food Donation Management
