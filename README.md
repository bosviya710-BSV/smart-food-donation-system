# 🍽️ Smart Food Donation System

A web-based application developed to reduce food waste by connecting food donors with NGOs and volunteers. The system helps distribute surplus food efficiently to people in need.

## 🚀 Live Demo
https://smartfooddonation.infinityfreeapp.com

## 📌 Features
- User Registration & Login
- Hotel Dashboard
- NGO Dashboard
- Volunteer Dashboard
- Admin Dashboard
- Food Donation Management
- Secure Authentication
